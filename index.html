Datefier

A simple plugin to create datepickers in few simple steps

Basic Usage

Import JQuery and JQueryUI:

<rel="stylesheet" href="//ajax.googleapis.com/ajax/libs/jqueryui/1.10.4/themes/smoothness/jquery-ui.css" />
<script src="//ajax.googleapis.com/ajax/libs/jquery/1.11.0/jquery.min.js"></script>
<script src="//ajax.googleapis.com/ajax/libs/jqueryui/1.10.4/jquery-ui.min.js"></script>

Import datefier.js 

<script src="path-to/datefier.js"></script>

Create an input field and give the datepicker class 

<input type="text" class="datepicker" />
Result:
click in the field

Other options

From - To

- Simply create a range from first date to last date 
<input type="text" class="datepicker from" />
<input type="text" class="datepicker to" />
Result:

Greater then today

- You can select only a date greater than today 
<input type="text" class="datepicker gt-today" />
or 
<input type="text" class="datepicker gte-today" />
Result:
or

Lower then today

- You can select only a date lower than today 
<input type="text" class="datepicker lt-today" />
or 
<input type="text" class="datepicker lte-today" />
